Symbol Saga Map Editor

Welcome to our tile based map generator. This gaming development tool is designed to allow creators to easily build maps for tile based games. To learn how to use our map editor please select a category below.

Select a category from the tile panel to see a list of available tiles. The tile you select will become highlighted, click on the blank map section to the right to place your tile. If you make a mistake, click on the highlighted tile to remove it from your current selection. Select the tile on the map that you would like to remove and push the "delete" or "backspace" key. Tiles can be laid on top of each other in any order you wish. Some tiles are part of bigger objects, so be creative and have fun building digital worlds!

The convert tool has two main purposes: Converting the current map in the editor to a numerical 2D array that can be added to the games code and rendering an exisiting map from code onto the map editor screen. Try an example by entering a number between 1 and 5 and pressing the "convert to map" button. Then press the convert to array button to see the numerical representation of that map.

The base tool allows for a quick start on building a new custom map. X represents the number of tiles wide you would like your map to be. Y represents the number of tiles in height you would like your map to be. Select a tile from the panel to automatically fill the tile field. Press the "Place Base" button and the editor will automatically render the base to the map.
